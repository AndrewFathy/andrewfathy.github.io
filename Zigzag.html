Simple Zigzag Game




Developed using Unity3D
Basic car game with an endless zigzag platform which is randomly generated, try to avoid falling off the platform and also stay away from the explosive wrecking balls.
Remember the ground tiles falls after a while from touching it.
This game illustrates good unity practices such as object pooling and Singleton pattern.
